# Getting Started

Base UI Vue provides unstyled, accessible UI primitives so you can build a design system with your own styles.

## Local Development

```sh
pnpm install
pnpm docs:dev
```

## Project Structure

- `docs/`: documentation source
- `packages/`: component source

## Next Steps

- Read the `Installation` guide
- Start writing component docs under `docs/components`
